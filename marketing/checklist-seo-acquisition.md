## 1. Google Search Console

À faire manuellement avec le compte Google rattaché au projet.

1. Aller sur Google Search Console.
2. Ajouter la propriété URL : https://marches-publics-martinique.vercel.app

## 2. Pages prioritaires à surveiller

### Page d’accueil

Objectif SEO : marchés publics Martinique, conseil marchés publics, commande publique Martinique.

Points à surveiller :
- titre principal clair ;
- proposition de valeur immédiate ;
- CTA diagnostic gratuit visible ;
- lien vers Insights ;
- lien vers Outils gratuits ;
- absence de promesse d’attribution.

### Entreprises privées

Objectif SEO : répondre appel d’offres Martinique, mémoire technique marché public, accompagnement appel d’offres.

Points à surveiller :
- CTA vers contact fonctionnels ;
- packs lisibles ;
- bénéfices formulés en obligation de moyens ;
- pas de garantie de résultat.

### Entités publiques

Objectif SEO : sécurisation DCE, accompagnement acheteur public, rédaction CCTP, analyse des offres.

Points à surveiller :
- cohérence juridique ;
- ton institutionnel ;
- preuve d’expertise achat ;
- CTA diagnostic clair.

### Insights

Objectif SEO : trafic long tail sur seuils 2026, MAPA, mémoire technique, prix anormalement bas, jurisprudence commande publique.

Points à surveiller :
- pages individuelles indexables ;
- JSON-LD Article présent ;
- maillage vers diagnostic ;
- sources officielles pour les contenus juridiques.
